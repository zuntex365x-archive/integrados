/////////////////////////////////////////////////////////////////////////////////////////
//
// Readme file for ComponentOne IntelliSpell
//
// Adds integrated spell-checking to Visual Studio 2008, 2010, 2012 and 2013. 
// 
// Spell-checking can be performed on individual files, projects, and 
// entire solutions.
//
// Spell-checks comments (including Xml documentation comments), strings, 
// localized resources, Html, Xml, Xaml, and text files. 
//
// Supports Word-style as-you-type checking.
//
/////////////////////////////////////////////////////////////////////////////////////////
